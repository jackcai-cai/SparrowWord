# SparrowWord 私隐政策

最后更新：2026-03-26

本政策说明 SparrowWord 如何处理你的资料。当前版本以本地学习工具为主；如未来启用网页账号、云端同步或服务器功能，我们会更新本政策。

## 1. 我们可能处理的资料

- 你输入的词汇、词组、句子及笔记
- 查词历史、复习记录、熟练度、收藏状态
- 应用设定，例如排序方式、离线资源状态
- 如果你主动启用相关功能，可能包括语音、截图识别文字或你自行输入的 API key

## 2. 当前版本如何储存资料

- 当前 macOS 版本主要将学习数据储存在你的本机装置上。
- 如果你输入 OpenAI API key，程式会优先尝试保存在系统 Keychain。
- 当前版本不会将你的本地学习数据自动上传给开发者自建服务器。

## 3. 第三方服务

- 如果你主动启用 AI 查词或相关联网功能，你的查询内容可能会发送到相应第三方服务提供者。
- 这些第三方服务有其自己的私隐政策和数据处理规则。
- 你应自行确认是否要启用该类功能。

## 4. 网页版与未来同步

- 如果未来版本加入网页端、账号系统或同步功能，我们可能会处理登录资料、同步后的词库数据及基础操作记录。
- 在这些功能正式开放前，我们会更新本政策并尽量用清楚方式提示你。

## 5. 我们如何使用资料

- 提供查词、复习、翻卡、进度记录及同步等功能
- 修复错误、提升稳定性及改善检索质量
- 在未来版本中防止滥用、作弊或未授权访问

## 6. 资料分享

- 除非法律要求，或为提供你主动启用的第三方服务所必需，我们不会主动向第三方出售你的个人资料。
- 我们不会把你的学习记录当作商品出售给他人。

## 7. 你的选择

- 你可以删除本地数据、停用联网功能、或不输入任何第三方 API key。
- 如果未来版本提供账号或同步功能，我们会尽量提供导出、删除或停用方式。

## 8. 儿童与学生

SparrowWord 面向学生和学习者，但不以收集敏感个人资料为目的。如你是未成年人，请在了解功能后再决定是否启用联网服务。
